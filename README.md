# TSLint Config kaiba

## Installation

```sh
npm install git+https://github.com/SpaceAgent-Inc/tslint-config-spaceagent --save-dev
or
yarn add --dev git+https://github.com/SpaceAgent-Inc/tslint-config-spaceagent
```

## Usage
In tslint.json:

```json
{
    "extends": "tslint-config-spaceagent/tslint"
}
```
